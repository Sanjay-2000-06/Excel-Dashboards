# HR Analytics Dashboard

 KPI-style HR dashboard exported to Excel, with quick analysis notes and recommended next steps for reproducible EDA and visualization.

---

## Files in this repo
- `HR ANALYTICS.xlsx` — Exported dashboard workbook (main sheet: `KPI`, 7 rows × 5 columns).  
- `hr_analysis_summary.txt` — short auto-generated summary of the workbook (contains sheet names and quick counts).  
- (Optional) `raw_hr_data.csv` — *recommended*: add a raw dataset file with row-level employee records if available.  
- `README.md` — this file.
